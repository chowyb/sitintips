Common Coding Mistakes

Several mistakes can be made by newer students who are unfamiliar with the intricacies of C and Java, or how functions provided by their respective libraries behave. Here are some possible mistakes to look out for.

  • Input

Reading String Inputs

Reading inputs is usually relatively easy, but when strings are involved things may be a little more complicated. Below are some ways to read in string inputs, and how they work.

Java

When mixing the nextLine() method with other methods from the Scanner class, note that while methods such as nextInt() and next() do not read in whitespaces such as line feed characters(\n), nextLine() does.
This may result in the case where using nextLine() immediately after a nextInt() or next() results in it reading only a line feed character, which is unintended.
To handle this, using a nextLine() call to consume the line feed character would work.
